# S21_SmartCalc_1.0.

Реализован калькулятор на языках C и C++ в проекте QT.
Реализованы дополнительные опции подсчета депозитного и кредитного калькуляторов. 
Депозитный калькулятор не работает идеально, связано это с отсутствием ввода Начала срока вклада. В задании это не требуется, но без этого точность вычислений несколько теряется из-за разного количества дней в месяцах, високосности года. Нельзя сделать более одного пополнения и снятия. Подсчет налогов будто отличается на сайтах banki.ru и calcus.ru, я ориентировалась на инструкцию с сайта calcus.ru (под надписью Налог гиперссылка "как рассчитывается").

## Установка

Чтобы установить приложение на компьютер, введите команду make install в директории src.
